ADMIN LOGIN GUIDE - HOW TO GET ADMIN TOKEN
==========================================

PROBLEM: User is on Admin Authentication page and needs to get Admin Token

SOLUTION: Use the simple admin login page that handles authentication automatically

LOGIN PROCESS:

1. ACCESS ADMIN LOGIN PAGE:
   URL: http://127.0.0.1:3006/admin_login_simple.html

2. ENTER CREDENTIALS:
   - Username: admin
   - Password: admin123
   - Email: admin@example.com (pre-filled)

3. AUTOMATIC TOKEN HANDLING:
   - Page calls backend API automatically
   - Receives JWT token from server
   - Stores token in localStorage
   - Redirects to admin dashboard

4. ACCESS ADMIN DASHBOARD:
   - After successful login, redirected to: admin_final.html
   - Token automatically included in all API calls
   - Dashboard loads with authenticated data

TECHNICAL PROCESS:

1. FRONTEND LOGIN:
   POST http://127.0.0.1:8000/admin/login
   {
     "username": "admin",
     "password": "admin123"
   }

2. BACKEND RESPONSE:
   {
     "access_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
     "token_type": "bearer",
     "expires_in": 3600,
     "admin_user": {
       "username": "admin",
       "role": "admin",
       "is_admin": true
     }
   }

3. TOKEN STORAGE:
   localStorage.setItem('admin_token', data.access_token)
   localStorage.setItem('admin_user', JSON.stringify(data.admin_user))

4. REDIRECT:
   window.location.href = 'admin_final.html'

ACCESS URLS:

1. ADMIN LOGIN PAGE:
   http://127.0.0.1:3006/admin_login_simple.html

2. ADMIN DASHBOARD (after login):
   http://127.0.0.1:3005/admin_final.html

3. BACKEND API:
   http://127.0.0.1:8000/admin/login

STEP-BY-STEP INSTRUCTIONS:

1. OPEN LOGIN PAGE:
   - Go to: http://127.0.0.1:3006/admin_login_simple.html

2. ENTER CREDENTIALS:
   - Username: admin (pre-filled)

4. DASHBOARD ACCESS:
   - Automatically redirected to admin dashboard

ALTERNATIVE: MANUAL TOKEN PROCESS

If you want to get token manually:

1. USE CURL:
   curl -X POST "http://127.0.0.1:8000/admin/login" \
        -H "Content-Type: application/json" \
        -d '{"username": "admin", "password": "admin123"}'

2. COPY TOKEN:
   Copy the "access_token" value from response

3. USE IN ADMIN_FINAL.HTML:
   - Go to: http://127.0.0.1:3005/admin_final.html
   - Enter the token in the "Admin Token" field
   - Click "Authenticate"

TROUBLESHOOTING:

1. LOGIN FAILS:
   - Check backend is running: python main.py
   - Verify admin user exists in database
   - Check network connectivity

2. TOKEN EXPIRED:
   - Login again to get new token
   - Token automatically refreshed on login

3. DASHBOARD NOT LOADING:
   - Check token is stored in localStorage
   - Verify backend endpoints are accessible
   - Check browser console for errors

The admin login page handles the entire authentication process automatically.
